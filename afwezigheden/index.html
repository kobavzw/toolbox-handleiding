Afwezigheden

Het is mogelijk om in Toolbox de afwezigheid van een leraar te registreren en verder op te volgen. De leraar kan zelf zijn afwezigheid aanvragen via de optionele module Mijn afwezigheden. De verantwoordelijke van de (personeels)administratie gebruikt hiervoor de module Afwezigheden. Deze fungeert als de beheersmodule waar men afwezigheden van leraren kan registreren, maar ook de aanvragen uit de module 'Mijn afwezigheden' kan opvolgen (goedkeuren of weigeren) en vervangers kan toewijzen. De afwezigheden stromen tevens door naar de module Vervangingslijsten voor het opstellen van de dagelijkse vervangingsroosters.
