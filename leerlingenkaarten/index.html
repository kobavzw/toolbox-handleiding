Leerlingenkaarten

Toolbox biedt de mogelijkheid om leerlingenkaarten aan te maken of te koppelen met de digitale leerlingenkaarten uit Smartschool. Op zo'n leerlingenkaart wordt een barcode of QR-code getoond. Deze code kan in Toolbox voor verschillende doeleinden gebruikt worden, zoals via de Wallet aankopen doen of leerlingen onder de middag scannen via de module Buitengaan.
