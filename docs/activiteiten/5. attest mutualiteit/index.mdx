---
title: Attest mutualiteit
slug: /attest_mutualiteit
---

Wanneer een activiteit werd aangevraagd via de activiteitenmodule, wordt er automatisch een attest aangemaakt voor elke deelnemer. Dat attest is terug te vinden in de module Leerlingenrekeningen bij het menu Rapporten. 

Klik [hier](/leerlingenrekeningen/Rapporten/#attest-mutualiteit-deelname-activiteit) voor meer info.
