# Archief


De archief pagina biedt een overzicht van alle webshops die momenteel in het archief staan.

### Filteren

Via de **Filter** kun je zoeken op de naam van een webshop.

### Webshop pagina
Klik op een rij in het overzicht om de [webshop pagina](/webshop/webshopPagina) van de betreffende webshop te bekijken.

### Webshops herstellen
![Webshop herstellen](webshops_archief_1.png)
klik op het herstellen icoon om de webshop terug in de lijst van de actieve webshops te plaatsen.
