In deze module krijg je een overzicht te zien van alle foutmeldingen die er zijn na het versturen van e-mails vanuit Toolbox, bv. via de module Leerlingenrekeningen, Openstaande facturen, Inventaris, … Op basis van deze foutmeldingen kan je achterhalen waarom bepaalde mails niet kunnen worden afgeleverd. Deze Toolboxmodule kan enkel gebruikt worden voor mails die verstuurd zijn via Amazon webservices en/of Toolboxmail.  

- Of je gebruik maakt van Toolboxmail kan je terugvinden in de module 'Instellingen > E-mail'.  
- In de module Leerlingenrekeningen kan je in het menu 'Scholen > E-mail/Smartschool/Parentcom' nagaan of je gebruik maakt van de Amazon webservices als volgende setting is ingevuld bij SMTP Server: ‘email-smtp.eu-west-3.amazonaws.com’.
