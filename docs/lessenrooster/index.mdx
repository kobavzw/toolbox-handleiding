Via de module Lessenrooster kan je het lessenrooster importeren vanuit de volgende roostersoftware: EDT, Mondriaan, Untis en MyRo. Na de import kan je vanuit Toolbox het lessenrooster importeren in Smartschool en ook meteen de juiste leerkracht aan de juiste klassen koppelen. Het lessenrooster is eveneens nodig om de module Vervangingslijsten te kunnen gebruiken.
