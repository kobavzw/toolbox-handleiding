Het is mogelijk om in Toolbox de afwezigheid van een leraar te registreren en verder op te volgen. De leraar kan zelf zijn afwezigheid aanvragen via de optionele module [Mijn afwezigheden](/afwezigheden/mijn_afwezigheden/). De verantwoordelijke van de (personeels)administratie gebruikt hiervoor de module [Afwezigheden](/afwezigheden/afwezigheden/). Deze fungeert als de beheersmodule waar men afwezigheden van leraren kan registreren, maar ook de aanvragen uit de module 'Mijn afwezigheden' kan opvolgen (goedkeuren of weigeren) en vervangingen kan toewijzen. De afwezigheden stromen tevens door naar de module [Vervangingslijsten](/vervangingslijsten) voor het opstellen van de dagelijkse vervangingsroosters.
