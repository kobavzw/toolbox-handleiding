In deze module kan je per dag een vervangingsrooster opstellen voor de afwezige leraren en/of klassen. Daarvoor wordt er informatie gehaald uit de module [Lessenrooster](/lessenrooster), aangevuld met de afwezigheden van leraren en eventueel gekoppelde vervangingen die geregistreerd werden in de module [Afwezigheden](/afwezigheden). Daarnaast is het mogelijk om ook rechtstreeks in deze module Vervangingslijsten afwezige leraren of klassen te registreren.
