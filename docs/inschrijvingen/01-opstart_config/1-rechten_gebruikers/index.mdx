---
title: Gebruikersrechten toekennen
---

Rechten voor een module worden toegekend via de module [Gebruikersbeheer](/gebruikersbeheer). De gebruikersrechten voor de module Inschrijvingen zijn opgesplitst in **Inschrijvingen beheer** en **Inschrijvingen gebruik**.

Voor de eindgebruiker is **Inschrijvingen gebruik** voldoende. Deze gebruikers kunnen nieuwe inschrijvingen ingeven, thuisinschrijvingen voltooien, eigen leerlingen herinschrijven, de inschrijvingen opvragen en opvolgingsacties toevoegen.
